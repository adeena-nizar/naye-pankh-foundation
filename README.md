# NayePankh Foundation Management System

A Flask-based web application for managing volunteers and donations for the NayePankh Foundation.

## Features

* Add Volunteers
* View Volunteers
* Search Volunteers
* Delete Volunteers
* Add Donations
* View Donations
* Filter Donations
* Generate Reports

## Technologies Used

* Python
* Flask
* CSV File Storage
* HTML
* Bootstrap
